🚀 Quick Start

Status: All systems ready! Choose your preferred method below:

⭐ Method 1: VS Code Live Server (RECOMMENDED)

1. Install "Live Server" extension in VS Code

2. Right-click index.html → "Open with Live Server"

3. Browser opens automatically to http://127.0.0.1:5500/

✨ Easiest method, auto-refreshes on changes

🐍 Method 2: Python HTTP Server

1. Open terminal in project directory

2. Run command:

python -m http.server 3000

3. Open browser to http://localhost:3000

📦 Method 3: Node.js Server

1. Open terminal in project directory

2. Run command:

node server.js

3. Open browser to http://localhost:3000

🪟 Method 4: Windows Batch File

1. Double-click start-server.bat

2. Server starts automatically

3. Open browser to http://localhost:3000

✨ Requires Python or Node.js installed
