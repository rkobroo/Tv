⭐ RECOMMENDED

🌟 VS Code Live Server

Best for development - auto-refreshes on changes

  1. Press Ctrl+Shift+X
  2. Type: Live Server
  3. Install by "Ritwick Dey"
  4. Right-click index.html
  5. Select "Open with Live Server"

✨ Opens at:

http://127.0.0.1:5500

🐍 Python Server

Works everywhere - if Python installed

  1. Open Terminal (Ctrl+`)
  2. Navigate to project
  3. Run command:
python -m http.server 3000

Open:

http://localhost:3000

📦 Node.js Server

Fast & reliable - if Node installed

  1. Open Terminal (Ctrl+`)
  2. Navigate to project
  3. Run command:
node server.js

Open:

http://localhost:3000

🪟 Windows Batch File

Easiest for Windows - one click!

  1. Find: start-server.bat
  2. Double-click it
  3. Terminal opens
  4. Server starts automatically

Open:

http://localhost:3000

⚙️ PowerShell Server

Windows built-in - no dependencies

  1. Open PowerShell
  2. Navigate to project
  3. Run command:
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process; .\http-server.ps1

Opens:

http://localhost:8000
