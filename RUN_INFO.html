📡 About This Application

Live Sports TV is a modern streaming application that allows you to watch HLS live TV channels from the iptv-org/iptv database. It features a responsive dark UI, smart channel filtering, and favorite management.

  • Live HLS streaming with Video.js player
  • Smart search and category filtering
  • Favorites management with localStorage
  • Regional support (NP, IN, BD, PK, LK, BT, etc.)
  • Automatic error recovery with retry logic
  • Responsive dark theme UI

🚀 How to Run Properly

For the best experience, you should run this application with a local web server. Choose one of the methods below:

1 VS Code Live Server (Recommended)

Install the "Live Server" extension, then right-click index.html and select "Open with Live Server". This will automatically open the app at http://127.0.0.1:5500

2 Python HTTP Server

Open terminal in project directory and run:

python -m http.server 3000

Then open: http://localhost:3000

3 Node.js Server

Open terminal in project directory and run:

node server.js

Then open: http://localhost:3000
